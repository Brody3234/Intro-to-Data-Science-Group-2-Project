# reports/

This folder is for exported report artifacts (e.g., HTML export of the notebook).

**How to generate (suggested):**
- In Jupyter: `File -> Export Notebook As -> HTML (.html)`
- Save the exported file here as `housing_factors_report.html` (or similar) for grading.

If the exported file is too large to commit, keep this README and submit the export through your course submission system instead.
